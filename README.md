# kalshi_bot
Python-based automated trading bot for Kalshi prediction markets using REST and WebSocket APIs.
Overview
Modular pipeline for ingesting prediction market data, testing trading strategies, and evaluating performance in a controlled environment.
Pipeline Components

Data ingestion via Kalshi REST and WebSocket APIs
Strategy logic and simulation layer
Performance evaluation framework (PnL, win rate, drawdown analysis)

Status
In progress. Core pipeline and data ingestion are functional. Strategy testing and evaluation framework in active development.
